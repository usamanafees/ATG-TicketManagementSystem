Email distribution to standard user roles for Task related events is determined by the values contained in the Task Default Email Notification List.

Standard user roles are:


Submitter: User (Caller) submitting the ticket.
Enterer: User entering the ticket
Assignee: User to whom the ticket is assigned
Next Action: User assigned NextAction on the ticket
Task Assignee: User assigned the task currently being added, assigned, or completed.
All Task Assignees: The group of all users assigned any task on the issue.
Task Completer: User completing the task.

ATG Extra supplies initial default values for these user roles. However, these values may be changed according to your needs. Be advised that these default values determine email notification for standard user roles on every Task.
