Email distribution to standard user roles for Ticket related events is determined by the values contained in the Ticket Default Email Notification List.

Standard user roles are:


All Task Assignees: The group of all users assigned any task on the Ticket
Assignee: User to whom the Ticket is assigned
Enterer: User entering the Ticket
Next Action: User assigned NextAction on the Ticket
Submitter: User (caller) submitting the Ticket

ATG Extra supplies initial default values for these user roles. However, these values may be changed according to your needs. Be advised that these default values determine email notification for standard user roles on every Ticket.
