User ID Maintenance Options

Read descriptions of the options below. Make a selection from the right of panel:


Merge Duplicate Users: Choose this menu option to combine two user records into one. Select the record you wish to retain (primary) and another User (secondary) to be replaced. This process will replace all instances of the secondary User with the primary User value you wish to retain, transferring all issues, assets, history records, etc. to the primary user. However, neither User record is modified. Check the box "Delete after processing" to remove the secondary user record from the system completely.
Change User's User ID: Choose this menu option to edit an existing User ID value. Enter the current User ID as well as the new value for User ID. The new value should not already exist in your Users table. This process will update the user record as well as any other instance of the ID found throughout the system (issues, assets, audit history, etc.
